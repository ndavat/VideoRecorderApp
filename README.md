````markdown name=README.md
# Video Recorder App

This application allows viewers to record videos with audio using either the front or back camera. Admins can log in to view and play the recorded videos. The application also has an option to store videos on cloud storage.

## Features
- Viewer login
- Admin login
- Record video with audio using front or back camera
- View and play recorded videos (Admin only)
- Option to map and store videos on cloud storage

## Requirements
- Android Studio
- Firebase Account (for Authentication and Cloud Storage)
````
